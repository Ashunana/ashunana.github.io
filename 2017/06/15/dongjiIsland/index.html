东极，岛歌

Jun 15, 2017 | 玩儿去 | Hits

某天，老茶、我、纶哥一块儿吃饭的时候，纶哥提议去东极岛领略祖国最最最最最东边的大好河山，饭太好吃了，没多考虑，东极岛之行一锤定音。
东极岛原来不是具体的某个岛屿，是一片岛屿的总称，比较有名的岛屿有三个，分别是：

  • 庙子湖：《后会无期》的主要取景地，发展最完善的一个主岛；
  • 东福山：最最最东边的一个有人居住的岛屿，超美，最接近水天一色的地方；
  • 青浜岛：以“海上布达拉宫”闻名，唔，还是陆上布达拉宫更好看。

    时间线拉到出行的第一天。

    第一天起了个大早，坐第一班地铁到达南京南，咻咻咻，南京南 → 宁波。

    # 宁波：南塘老街、天一阁

    南塘老街的建筑以明清的江南民居风格为主，和蓝鲸的夫子庙、老门东大同小异。随便吃了些便前往下一站了。
    天一阁是中国现存最早的私家藏书楼，还是亚洲最古老的图书馆，建于明朝中期，是由当时的兵部右侍郎范钦主持建造的。据说，天一阁的建筑格局、防蛀护书的方法独具一格，并被后来人借鉴。看着不大，然而东绕西绕的也能走到腿酸。
    古人认路的本领不是盖的。
    时间不早了，坐上大巴，晃晃悠悠，宁波 → 舟山
    第一次坐2h的大巴，还是跨海的，还是坐在驾驶室上方的第一排（假装是自己用意念开的车，服气！）。途径舟山跨海大桥，Wua~，天气好棒，后方的落日，前方的云，不可方物。

    # 舟山：可以看到海的房间

    下了大巴，出了站。
    胖友，你见过7:30P.M.已经打烊的McD吗？还有空荡荡的街道。
    胖友，你体验过24层的电梯惊魂吗？熏疼自己。

    但素，这波体验我给满分，因为太美妙——深夜的海岸和清晨的窗景。

    时间线缓缓拉第二天惹

    依旧起了早，来到沈家门码头，东极岛只能坐船去，┑(￣Д ￣)┍。因为事先在万能的淘宝上订好了船票，到达码头只需要打电话给客服取票就好。
    太阳慢慢升起，轮船划开海面，舟山群岛 → 东极
    中途大船会停两次，因为想前往东福山岛，所以上了船先补票，然后到最后再下船。

    船程：2.5h。一开始老激动啦，哇海！哇！！海水变蓝啦！！！哇！！！！看不到岛了！！过了多久了？什么才一个小时？海面空荡荡···咦飘过去了一个木板！！海面空荡荡····快靠岸了快靠岸了！！！灯塔灯塔！！！有人在礁石上钓鱼耶！！！换船啦换船啦，还有半小时~~~到了呢！！！

    # 东福山岛：海角1号、徒步环岛

    踏上陆地的第一感觉就是：踏实。虽然算命的师傅说我命里缺水，但是一下子被水包围也不是很舒服的说。上岛后就有热情的阿姨（后证实原来就是预定的旅馆的店员）指路，放眼过去，都是住宿的地。预定的旅馆名字是海角一号，老板娘奉行先到先得的原则（嘻嘻），我们挑了2楼的两间，有水有电还能开空调，一本满足。

    关于吃饭吃什么的大问题，答案是旅馆的阿姨那儿点菜就好啦，除了价格比普通小饭馆贵一点之外没有什么缺点，一本满足。环山公路起点处有排大排档，唔，还是回去吃烧烤吧。

    吃饱睡足，下午3点的样子开始了顺时针的徒步环岛，无以言的美以及翻过一个山头还有一个山头的酸爽，捂脸哭。

    礁石、海浪、石阶、同行的妹纸、叫黑龙的拉布拉多，啊还有落日——只是近黄昏的兴叹是不存在的。

    近7点的时候我们终于回到了住宿地，超应景的可乐，蓝后洗个澡吃碗海鲜面，饭后溜达看到疑似发光的水母群，今日份的岛主扮演至此终结。

    时间线又开始动啦停在第三天

    第三天是不带脑子（前两天带了？）不断换乘交通工具的奇妙一天。
    早起在新东方第一缕曙光观测台欣赏了一场因为云不存在的日出，回去睡了个回笼觉就坐上了最早的一班船前往了庙子湖岛。

    # 庙子湖岛：后会无期、自由男神像

    因为距离前往舟山的船要过好久才会开，所以按照计划我们坐上了环岛观光车。中途开车的师傅还来了一段少老不宜的过山车（心辣么大）。值得一提的是自由男神像（其实是财伯公塑像——为渔民无私举火指航的大好人）旁边的观景台，私心认为这边风景独好。

    日照杆头，轮船再次划开海面，东极 → 舟山群岛。
    日落时分，坐上大巴，晃晃悠悠，舟山 → 宁波。
    缓缓入夜，高铁在飞，宁波 → 杭州东。
    入住杭州下城区。

    时间线直奔第四天

    # 杭州：西湖、岳庙、灵隐寺、武林广场

    早上睡得超饱。

    滴滴滴西湖的水呀晃眼睛，可乐味的碎冰冰❤。
    坐上游船到达湖中央，白天依旧不会存在的三潭印月（有机会还真想验证下古会玩）。
    灵隐寺的2岁小和尚呀，真希望你未来拥有的是和大堂里面诵经师傅们不一样的素淡的世故和明白的愚。

    晚上和老茶两个人逛了好久的街，芒果雪花冰塔满分。

    时间线滑到最后一天了呢

    ☆ 杭州：南宋御街、胡雪岩旧居

    南宋御街——再也不信什么老街惹，如果小吃好吃的话还能在老街中占一席之地。

    距离老街不远的地方有一处府邸——是胡雪岩居住了不到10年的豪宅。

    粗略逛下来，想着住在这样的宅子里面真舒服，然而胡雪岩住了不到十年，便郁郁而终了。

逛完最后一站便归宁啦。
五日最佳的是：走到哪儿都是好天气~躲过了蓝鲸的暴雨~也躲过了杭州的倾缸大雨！感觉自带吸太阳体质
一路不带脑子倍感轻松，感谢老茶的体贴照顾，感谢纶哥规划的10分满分的10分的行程，感谢古会玩！
